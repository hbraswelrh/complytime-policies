# ComplyTime Policies Resources

Start here to find documentation for the ComplyTime ecosystem.

---

### Getting Started

If you're new to the project, we recommend reading in this order:

1. **[Common Terms](COMMON-TERMS.md)** — glossary of terms used across ComplyTime and Gemara, grouped by topic. Start here to build a shared vocabulary.
2. **[Gemara Lexicon](gemara-lexicon.md)** — formal definitions of Gemara-specific terms, organized by the 7-layer model.
3. **[complyctl Overview](complyctl-overview.md)** — overview of the complyctl CLI tool, with walkthroughs for engineers, compliance managers, and auditors.
4. **[FAQ](FAQ.md)** — frequently asked questions about ComplyTime, Gemara, policies, and providers.
